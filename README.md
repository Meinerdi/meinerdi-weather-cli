# Weather CLI

**Weather CLI** is a simple command-line tool for fetching weather forecasts.

## Installation

To install the CLI globally using npm, run:

```bash
    npm install -g meinerdi-weather-cli
```

## Commands:

### Get the current weather

```bash
    node weather
```

### Get help

```bash
    node weather -h
```

### Set city

```bash
    node weather -c "city"
```

### Set token

```bash
    node weather -t "token"
```
